sdvmodman

sdvmodman is a suckless-style mod manager for Stardew Valley intended to be a lightweight solution for advanced users. It is written in C and is configured at compile time through config.h.

Running sdvmodman will launch the interactive CLI interface allowing one to toggle each installed mod between enabled and disabled. Running sdvmodman -i <path to mod> will install the mod specified.
